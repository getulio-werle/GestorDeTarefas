# Gestor de Tarefas

## Descrição
Aplicação web para gerenciamento de tarefas. Foi desenvolvida durante um curso de Desenvolvimento Web para conhecer o framework CodeIgniter.

## Tecnologias utilizadas
- HTML
- CSS
- Bootstrap
- JavaScript
- PHP
- CodeIgniter
- Biblioteca DataTables
